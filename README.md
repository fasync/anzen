# Anzen

Anzen is a minimal and simple password generator.

## Usage
### Example
```anzen -cnsl 32``` \
```anzen -l 64``` \
```anzen -cnl 16```

### Required
```-l``` specify the length of the password

### Optional
```-c``` will include capitalized characters \
```-n``` will include numbers \
```-s``` will include special characters like !")$% etc.
